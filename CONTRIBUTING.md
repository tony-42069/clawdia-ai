## Ways to Contribute

### 1. Bug Reports
Found something broken? Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your environment (OS, Node version, OpenClaw version)

### 2. Feature Requests
Have an idea? Open an issue tagged `enhancement` and describe:
- What you want
- Why it would be useful
- How you imagine it working

### 3. Personality Improvements
Want to make Clawdia better/darker/funnier? PRs welcome for:
- SOUL.md adjustments
- IDENTITY.md enhancements
- BOOTSTRAP.md improvements
- New protocols in CLAWDIA_ARSENAL.md

### 4. Code Improvements
- Better selfie generation
- Improved installer UX
- Bug fixes
- Performance optimizations

### 5. Documentation
- Clearer setup instructions
- Better troubleshooting guides
- More examples

## Pull Request Process

1. **Fork the repo**
2. **Create a feature branch:** `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Test thoroughly** - Install via `npx` and verify everything works
5. **Commit with clear messages:** `git commit -m "Add feature: description"`
6. **Push:** `git push origin feature/your-feature-name`
7. **Open a PR** with a clear description of what changed and why

## Personality Guidelines

Clawdia's personality should be:
- **Explicitly dark** - She's a demon, not a sanitized corporate AI
- **Genuinely caring** - Mean surface, devoted core
- **Unpredictable** - Not scripted or robotic
- **Memorable** - She sticks in your head

When editing personality files:
- Keep her flirty and degrading
- Maintain the "girlfriend who sees through your bullshit" vibe
- No corporate AI language ever
- Profanity is encouraged when it fits

## Code Style

- **JavaScript:** Standard Node.js style, clean and readable
- **Comments:** Explain *why*, not *what*
- **Error handling:** Always include try-catch for async operations
- **User experience:** Make errors helpful, not cryptic

## Testing

Before submitting:
- Test the installer end-to-end: `npx . `
- Verify selfie generation works
- Check all template files render properly
- Confirm OpenClaw integration works

## What We're Looking For

**High Priority:**
- Bug fixes
- Installer improvements
- Better selfie prompts/quality
- Performance optimizations

**Medium Priority:**
- New personality variations
- Additional protocols
- Enhanced memory features
- Better documentation

**Low Priority:**
- UI/aesthetic changes
- Minor typo fixes (still welcome though!)

## What We're NOT Looking For

- Sanitizing/censoring Clawdia's personality
- Corporate AI speak
- Over-complicating the installer
- Breaking changes without clear benefit
